# 注册与登录

首次使用 KernelGen 时，请执行以下步骤：

1. 在浏览器中打开 <https://KernelGen.flagos.io/login>。
2. 点击 **进入工作台**。

    ![alt text](../assets/images/1-workspace-ch.png)

1. 选择通过手机号或电子邮箱注册：

    {style=lower-alpha}
    1. 在 **登录/注册** 对话框中，点击 **手机号登录** 或 **邮箱登录**。

        ![alt text](../assets/images/2-login-and-signup-ch.png)

    2. 阅读并勾选底部的复选框以接受 **用户协议** 和 **隐私政策**。
    3. 输入您的手机号或电子邮箱地址，然后点击 **获取验证码**。
    4. 通过短信或电子邮件收到验证码后，将验证码复制并粘贴到您的手机号或电子邮箱地址下方，然后点击 **登录/注册**。
    5. 输入您的姓名、组织、电子邮箱地址或手机号以及使用目的，然后点击 **提交**。

        ![alt text](../assets/images/3-fill-personal-information-ch.png)

    欢迎页面将会打开。
